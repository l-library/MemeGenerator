# 表情包生成器
一款轻量、易操作的表情包生成工具。  
## 开发环境
- Qt Creator
- Qt 6.10
- MinGW
## 开发计划
1. 图片相关
   - [X] 导入图片
   - [X] 多图叠加
   - [ ] 背景去除
   - [ ] 滤镜添加
   - [ ] 裁剪与缩放
   - [ ] 赛博做旧
2. 文本相关
   - [X] 添加文本
   - [X] 文本编辑
   - [ ] 字体自定义
   - [ ] 文字气泡、对话框模板。
3. 绘图工具
   - [ ] 涂鸦功能
   - [ ] 插入箭头、线条等形状
   - [ ] 马赛克效果
4. 模板功能
   - [ ] 常见表情包模板导入
   - [ ] 自定义模板，批量生成固定模板表情包
5. 个性化与实用功能
   - [ ] 多图拼接
     - [ ] 所有图片展示全貌，适用于漫画/连环画型表情包
     - [ ] 仅第一张图展示全图，剩余自定义只展示一部分，适用于切割字幕型表情包
   - [ ] 一键调整尺寸以适配特定平台
   - [ ] 水印/签名添加
6. 更多进阶功能……
